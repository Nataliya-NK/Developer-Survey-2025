# Developer-Survey-2025
Stack Overflow Developer Survey 2025 — Аналіз даних
Цей проєкт містить аналіз опитування Stack Overflow Developer Survey 2025. Мета — дослідити профіль сучасних розробників, їх технології, освіту, досвід роботи, рівень компенсації та інші особливості, використовуючи інструменти Python.
Notebook включає очищення даних, обробку, групування, побудову графіків та формування узагальнюючих висновків.
 Використані технології
•	Python
•	Pandas — обробка даних
•	NumPy — числові операції
•	Matplotlib— візуалізація
•	Jupyter Notebook — середовище виконання
Структура проєкту
•	survey_analysis.ipynb — основний ноутбук з усім аналізом
•	README.md — опис проєкту
Основні результати аналізу

 1. Респонденти та структура опитування
•	Загальна кількість учасників: 49 191
•	Кількість запитань у schema: 139
•	Жоден респондент не відповів на всі запитання повністю
(значна кількість пропусків — характерна риса цього опитування).
2. Досвід роботи (Work Experience)
•	Діапазон значень: від 1 до 100 років
•	Присутні викиди, які впливають на розподіл
(наприклад, значення 80–100 років досвіду).
3. Формат роботи
•	Кількість офлайн (офісних) працівників дещо вища, ніж кількість віддалених.
•	Віддалена робота все ще популярна, але не домінує.
4. Використання Python
•	37,54% усіх опитаних програмують на Python.
•	Аналіз за віковими групами показав:
o	Найбільша частка Python-розробників — у групі 18–24 роки.
o	Найменша — серед розробників 55+ років.
5. Освіта та компенсація
•	Спеціалісти з магістерським ступенем заробляють у 3–5 разів більше, ніж власники бакалаврського.
•	Магістратура є одним з найсильніших факторів високої зарплати в опитуванні.
